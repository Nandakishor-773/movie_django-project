In the world of endless movie choices, this web application is designed for sharing, discovering, and discussing the latest movies.  
The platform allows users to contribute by adding new movies, sharing their thoughts through comments, and building a personalized list of favorites.
This project is built entirely using Django and demonstrates backend development, user authentication, database management, and role-based access control.

Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS
- Database: SQLite
- Authentication: Django Auth
- Version Control: Git & GitHub
